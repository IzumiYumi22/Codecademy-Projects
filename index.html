HTML & CSS Helpful References!

HTML Tags

Tag Discription
<!DOCTYPE> Defines the document type.
<html> Defines an HTML document.
<head> Contains metadata/information for the document.
<title> Defines a title for the document.
<body> defines the document's body.
<h1-h6> Defines HTML headings. H1 is the largest heading. H6 is the smallest.
<p> Defines a paragraph.
<img> Defines an image.
<figcaption> Defines a caption for the <figure> element.
<figure> Specifies self-contained content.
<a> Defines a hyperlink.
<link> Defines the relationship between a document and an external resourse.
<nav> Defines navigation links.
<div> Defines sections of a document.
<header> Defines a header for a document or section.
<footer> Defines the footer of a document or section.

CSS Properties

Properties Description Example
Margin Sets all the margin properties in one declaration. Can use with Top, Bottom, Left, Right. header { margin-top: 0px; Margin-bottom: 8px }
Position: static The static position is the default position in the document flow.
Position: absolute Position relative to ancestor absolute element.
Position: relative Relative to default posistion.
Position: fixed Fixed location on the screen.
Color Sets the foreground color value of an element's text and text decorations, and sets the currentcolor value. header { color: blue; }
Border Serts all the border properties in one declaration. You can do border-width (sets the thickness of the border box.), border-style (sets the style e.i. dotted, solid.) and border-color (which sets the color of the border box.) header { border-width: 2px; border-style: dotted; border-color: blue }
Height Sets the height of an element. header { height: 50px; }
Width Sets the width of an element. header { width: 50px; }
Padding Used to generate space around an element's content, inside of any defined borders. Can do top, bottom, right, left. header { padding: 1px, 0, 3px; }
Align-content Sets the distribution of space between and around content items along a flexbox's cross axis or a grid or black-level element's black axis. header { align-content: center; }
